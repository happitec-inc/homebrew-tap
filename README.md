# happitec-inc/homebrew-tap

Homebrew tap for [happitec-inc](https://github.com/happitec-inc) command-line tools.

## Install

```
brew tap happitec-inc/tap
brew install happitec-inc/tap/<formula>
```

## Formulas

### pushover-notify

Send Pushover notifications with optional image attachments.

Source: [happitec-inc/pushover-notify](https://github.com/happitec-inc/pushover-notify)

```
brew install happitec-inc/tap/pushover-notify
```

### gh-filter

Allowlist shim for the gh CLI that refuses out-of-org invocations.

Source: [happitec-inc/gh-filter](https://github.com/happitec-inc/gh-filter)

```
brew install happitec-inc/tap/gh-filter
```
